## 📱 具体改进

### 1. 背景色 - 活力米白
```swift
// 之前：深色背景
Color(red: 0.11, green: 0.11, blue: 0.12)  // 接近黑色

// 现在：暖米白色
Color(red: 0.97, green: 0.95, blue: 0.92)  // 温暖米白
```

### 2. 文字颜色 - 深色清晰
```swift
// 之前：白色文字（在深色背景上）
textPrimary: Color(1.0, 1.0, 1.0)  // 纯白

// 现在：深色文字（在浅色背景上）
textPrimary: Color(0.20, 0.20, 0.22)  // 深炭黑
```

**对比度提升**：从白色文字到深色文字，在浅色背景上更加清晰可读！

### 3. 卡片背景 - 层次分明
```swift
// 普通卡片：浅米白色
cardBackground: Color(1.0, 0.98, 0.95)

// 提升卡片：纯白色（更亮）
cardBackgroundElevated: Color.white
```

### 4. 粒子效果 - 深色粒子
```swift
// 之前：白色粒子（在深色背景上）
.fill(Color.white.opacity(0.25))

// 现在：深色粒子（在浅色背景上）
.fill(theme.textTertiary.opacity(0.15))
```

**效果**：灰色粒子在米白色背景上漂浮，优雅不突兀

### 5. 强调色 - 更鲜艳
```swift
// 紫色主色
primaryAccent: Color(0.55, 0.35, 0.85)  // 深紫色

// 蓝色次色
secondaryAccent: Color(0.30, 0.55, 0.95)  // 鲜蓝色

// 收入绿
incomeGreen: Color(0.30, 0.70, 0.45)  // 鲜绿色

// 支出红
expenseRed: Color(0.95, 0.30, 0.35)  // 鲜红色

// 警告黄
warningYellow: Color(1.0, 0.70, 0.20)  // 鲜黄色
```
